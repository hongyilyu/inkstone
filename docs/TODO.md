- [ ] Click-to-refocus may not work in all terminal emulators
- [ ] pi-agent-core message history not restored on session load (only display messages)

- [ ] `DialogSelect` has no grouped-category or gutter rendering yet ‚Äî categories on `DialogModel` options and `"‚úì Connected"` on `DialogProvider` currently surface through the `description` field.
